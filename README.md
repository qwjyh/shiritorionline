# about this
This is a bot for LINE which judge shiritori(a word chain game).

# これは何
しりとり判定bot on LINE

ベースは[これ](https://github.com/qwjyh/flaskshiritori)

トーク、グループ、トークルームごとにしりとりの判定をしてくれる(ただし、ひらがなのみ、もしくは片仮名のみ限定)
